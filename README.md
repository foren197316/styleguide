# InterExchange Styleguide

```
bundle exec foreman start
```

## Installing

### `head`

```
<head>
  <meta charset="utf-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link href="https://styleguide.interexchange.org/images/logo-100.png" rel="shortcut icon" type="image/vnd.microsoft.icon" />
  <title>InterExchange</title>

  <link href="https://styleguide.interexchange.org/stylesheets/interexchange.css" rel="stylesheet" type="text/css" media="screen, print" />

  <!--[if lt IE 9]>
    <script src="//oss.maxcdn.com/html5shiv/3.7.2/html5shiv.min.js"></script>
    <script src="/javascripts/respond.min.js"></script>
  <![endif]-->
</head>
```

### `foot` sits at the bottom of the html tag

```
<html>
  <head>...</head>

  ...

  <script src="/javascripts/interexchange.js" type="text/javascript"></script>
</html>
```
